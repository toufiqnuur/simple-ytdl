# ytdl
Simple Youtube Video Downloader built with ♥️ and nodejs.

# install
```
git clone https://github.com/toufiqnuur/simple-ytdl.git
```

Masuk ke direktori ytdl
```
cd simple-ytdl/
```

Install module
```
npm install
```

Jalankan Server
```
node index.js
```

Server running on port 3000
```
localhost:3000
```

Good luck!
